# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-01-03

### Added
- Initial release of `m4b-joiner.py`.
- Functionality to join multiple MP3 files into a single M4B file.
- Chapter support based on input filenames or a custom order file.
- Cross-platform check for `ffmpeg` and `ffprobe` availability.
- Strict validation ensuring all input files share the same sample rate and channel count.
- Documentation (`README.md`) and requirements (`requirements.txt`).
